# ERP Demo Starter

Next.js starter for an ERP capability demo with:

- A landing page
- An interactive ERP demo dashboard
- Role login pages for `stock-manager`, `sales-agent`, `client`, and `owner`

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- The original static prototype is kept in [`index.html`](./index.html) as a reference source.
- The active app lives in `app/`, `components/`, and `data/`.
- Login forms are static placeholders ready for real auth wiring.
